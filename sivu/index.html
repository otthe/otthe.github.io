Sivu

Sivu is a server-side templating DSL built on top of Express.js that lets you write backend JavaScript directly inside HTML using PHP-style syntax.

Syntax


<!-- index.sivu -->

<?sivu
  const { formatDate } = await $import("./format.js");
  let title = "Sivu Todo Example";
  const todos = await db.query("SELECT * FROM todos");
?>

<?include "_header.sivu"?>

<script nonce="<?= $nonce(); ?>">
  console.log("executed on client!");
</script>

<form method="POST" action="/add_todo.sivu">
  <?= $csrf($_SESSION) ?>
  <label>Task name</label><br>
  <input type="text" name="taskname"><br>
  <button>Add Todo</button>
</form>

<?sivu for (const todo of todos) { ?>
  <h4><?= todo.task_name ?></h4>
  <?= formatDate(todo.due) ?>
<?sivu } ?>

Built-in Functions

All built-in functions are prefixed with $ to avoid naming conflicts with user-defined variables and JavaScript globals.

HTML & Output

$echo(...values)

Outputs one or more values. Each value is automatically escaped unless marked as safe HTML. 


    $echo("Hello", " ", "<b>World</b>")
    // Output: Hello <b>World</b>

$print(value)

Outputs a single value (alias of $echo). Returns 1. 


    $print("Hello")
    // Output: Hello

$println(...values)

Outputs values followed by a newline (\n). 


    $println("Hello")
    // Output: Hello\n

$html(value)

Marks a string as safe HTML (no escaping).


  $html("<b>Hello</b>")

$raw(value)

Alias for $html().

$escapeHtml(value)

Escapes HTML characters.

$htmlEntities(value)

Encodes characters into HTML entities.

$dump(value)

Pretty prints a variable for debugging.

$yield()

Outputs layout content.

Database

$connect(type, options)

Connects to a SQLite database.


  const db = $connect("sqlite", { file: "app.db" })
  db.query("SELECT * FROM users")

Security (CSRF)

$csrf()

Generates a hidden CSRF input field.

$generateCsrfToken(session)

Generates a CSRF token.

$verifyCsrfToken(session, token)

Validates a CSRF token.

Flash Messages

$flash(key, value)

Stores a flash message.

$flashGet(key)

Gets and removes a flash message.

$flashPeek(key)

Gets a flash message without removing it.

$flashAll()

Returns all flash messages and clears them.

Response Helpers

$status(code)

Sets HTTP status code.

$header(name, value)

Sets a response header.

$contentType(value)

Sets Content-Type header.

$json(data, code)

Sends a JSON response and exits.

$send(body, code)

Sends a raw response and exits.

Redirects

$redirect(url, status)

Redirects to a URL.

$back()

Redirects back (fallback supported).

Session

$sessionExists()

Checks if session exists.

$sessionId()

Returns session ID.

$sessionRegenerate()

Regenerates session ID.

$sessionDestroy()

Destroys session.

$sessionSave()

Saves session.

$sessionReload()

Reloads session.

Misc

$nonce()

Returns CSP nonce.

$import(path)

Imports a module (ESM).

$die(message)

Stops template execution.

$exit(message)

Alias for $die().

Superglobals

These variables are always available inside templates.

$_GET

Query parameters from URL.


// /page?name=John
$_GET.name // "John"

$_POST

POST request body.

$_SESSION

Session data storage.


$csrf($_SESSION)

$_COOKIE

Cookies (requires middleware).

$_ENV

Environment variables.

$_SERVER

Request metadata.


$_SERVER.requestMethod
$_SERVER.requestUri
$_SERVER.httpHost
$_SERVER.httpUserAgent
$dump($_SERVER)

Things to consider

Sivu is in a very early stage and might contain some bugs and security issues. Also this documentation might not always be 100% accurate.
